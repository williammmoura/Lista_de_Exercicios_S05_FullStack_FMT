Exercícios da Semana 05 - FullStack (FMT/SENAI-SC)

Exercício 01

Faça uma arrow function para retornar uma mensagem de "Olá, Mundo!". Armazene esta arrow function em uma constante (const) de nome "mensagemOla". Após a definição, execute a função "mensagemOla".

Exercício 02

Ajuste a função "mensagemOla", criada no exercício anterior, para que receba um parâmetro "nome" e retorne a mensagem “Olá, [nome]!” (ex.: Olá, Ana!). Após o ajuste, execute a função "mensagemOla", mas dessa vez passando algum nome como parâmetro.
